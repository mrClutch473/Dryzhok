# 📌 О проекте

**Дружок** — это полноценный веб-сайт для приюта бездомных животных, разработанный на Django.

Сайт позволяет пользователям познакомиться с подопечными приюта, узнать подробности о каждом животном и связаться с организацией для усыновления. Администраторы управляют контентом через встроенную Django Admin панель — без необходимости писать код.

Проект реализует полный цикл разработки: спроектированы модели данных, настроена маршрутизация URL, реализованы серверные представления и написан адаптивный фронтенд.

---

# ⚙️ Основные возможности

## Каталог животных

Страница со всеми питомцами приюта, отображёнными в адаптивной сетке карточек. Каждая карточка содержит фотографию, имя, возраст, пол и вид животного.

---

## Фильтрация без перезагрузки страницы

Пользователь может отфильтровать питомцев по категориям прямо на странице — без обращения к серверу:

* Все
* Собаки
* Кошки
* Мальчики
* Девочки

---

## Модальные окна с подробной информацией

При нажатии на карточку открывается модальное окно с полным описанием животного, его фотографией и кнопкой «Хочу забрать домой».

---

## Адаптивный дизайн

Вёрстка корректно отображается на всех устройствах:

* Десктоп — 4 колонки
* Планшет — 3 колонки
* Мобильный — 1–2 колонки

---

## Навигация по разделам

Сайт включает следующие разделы:

* Главная
* О приюте
* Наши животные
* Помощь приюту
* Новости
* Контакты
* Отчётность

---

## Административная панель

Контент полностью управляется через стандартную Django Admin панель: добавление, редактирование и удаление карточек животных без вмешательства в код.

# 🧩 Модель данных

Основная модель `Animal` хранит всю информацию о питомце:

| Поле | Тип | Описание |
|---|---|---|
| `name` | CharField | Имя животного |
| `age` | CharField | Возраст |
| `pol` | CharField (choices) | Пол: `Men` / `Women` |
| `tip` | CharField (choices) | Вид: `Dog` / `Cat` |
| `text` | TextField | Описание животного |
| `photo` | ImageField / URLField | Фотография |

# 🔑 Управление контентом

Для добавления нового питомца:

1. Войдите в Admin панель: `https://dryzhok.ru/admin/`
2. Перейдите в раздел `Animals`
3. Нажмите `Добавить`
4. Заполните поля: имя, возраст, пол, вид, описание, фото
5. Сохраните запись — питомец сразу появится на сайте
